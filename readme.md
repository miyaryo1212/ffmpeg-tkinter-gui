# FFmpeg GUI

## 概要

話題のGPT-4oを使ってコーディングしました。GUIからFFmpegを呼び出すソフトです。実装にはPython標準のTkinterを使っているため特にインストールなしで使えます。

## 使い方

FFmpegの標準的なインストールとPathが通っていることを確認してください。このプログラム自体は特に処理をせず、インストール済みのFFmpegを呼び出すのみです。
